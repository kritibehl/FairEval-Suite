# Public Benchmark: Instruction Following

This benchmark compares multiple LLMs on a focused instruction-following task set and evaluates whether a candidate model should ship or be blocked.

## Models
- GPT-4o-mini
- Claude Haiku
- Gemini Flash

## Artifacts
- raw run reports
- compare artifacts
- gate artifacts
- regression case library

## Goal
Turn FairEval from an evaluation framework into a verifiable release-gating system with public benchmark evidence.
