# LLM Regression Gate Demo — Instruction Following Benchmark

This benchmark package is the public proof layer for FairEval.

It compares multiple LLMs on a focused instruction-following task set and evaluates whether a candidate model should ship or be blocked.

## Models
- GPT-4o-mini
- Claude Haiku
- Gemini Flash

## Artifacts
- raw run reports
- compare artifacts
- gate artifacts
- regression case library
- benchmark summary

## Goal
Turn FairEval from an evaluation framework into a verifiable release-gating system with public benchmark evidence.

## Public framing
FairEval is not just scoring model outputs.

It is testing release readiness.
