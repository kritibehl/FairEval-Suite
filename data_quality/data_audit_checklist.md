# Data Audit Checklist

## Benchmark integrity
- duplicate case IDs checked
- duplicate payloads checked
- schema validity checked
- missing expected IDs checked

## Annotation quality
- labeling consistency reviewed
- disagreement cases reviewed
- failed constraints documented
- ambiguous cases flagged

## Dataset completeness
- required fields present
- prompt versions recorded
- evaluator versions recorded
- dataset hashes generated

## Regression review
- failed cases exported
- release-gate artifacts generated
- compare reports validated
- downstream risk reviewed
