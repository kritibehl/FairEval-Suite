# Eval Run Comparison UI

FairEval includes a React dashboard for comparing evaluation runs and release-safety signals.

## UI views
- baseline vs candidate quality comparison
- Responsible AI release gate summary
- RAG groundedness and hallucination checks
- latency / cost threshold gate
- oversight reliability comparison
- evaluation warehouse summary

## Source
`dashboard_frontend/`

## Safe scope
This is an interactive local dashboard over FairEval artifacts, not a production analytics product.
