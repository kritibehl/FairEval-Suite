# FairEval Dashboard Export

FairEval exports benchmark results into dashboard-ready reports for model benchmark review.

## Files
- `exports/faireval_results.csv`
- `exports/faireval_summary.json`
- `exports/dashboard_screenshot.png`

## Fields
- model name
- benchmark suite
- average score
- pass rate
- failed case count
- variance metrics
- gate decision
- release outcome

## Why this matters
This gives ML platform and release reviewers a self-service view of model quality, variance, failed cases, and release-gate outcomes.
