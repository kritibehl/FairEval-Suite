## Benchmark v1: AI Release Safety Benchmark

FairEval Benchmark v1 evaluates GPT/Claude/Gemini-style text outputs and agentic workflows across:

- groundedness
- hallucination / unsupported-answer risk
- Responsible AI false allows
- refusal and escalation consistency
- latency and cost/request thresholds
- baseline-vs-candidate regressions
- oversight evaluator reliability
