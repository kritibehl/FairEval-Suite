# Evaluation Job Dashboard

FairEval includes a lightweight evaluation-job dashboard generated from async worker artifacts.

## Dashboard fields
- queued jobs
- completed jobs
- failed evaluations
- retry jobs
- review-before-release count

## Why this matters
Async evaluation systems need operational visibility. This dashboard turns worker results into a simple review artifact for evaluation operations and release workflows.

## Safe scope
This is a static dashboard artifact generated from local worker output, not production monitoring infrastructure.
