# NLP Evaluation Metrics

FairEval includes a small NLP evaluation metrics artifact for model-output analysis.

## Metrics computed
- precision
- recall
- F1
- confusion matrix
- accuracy

## Source
Metrics are computed from `reports/nlp_eval_report.json`, which compares expected issue categories against predicted issue categories for model-output failure examples.

## Why this matters
These metrics demonstrate ML evaluation fundamentals and support model-output analysis workflows.

## Safe scope
This is a small NLP evaluation demo, not a production model benchmark or training pipeline.
