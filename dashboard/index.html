FairEval Reliability Dashboard

Version comparison, drift alerts, rollback recommendation, and recent traces.

Recommended API flow

dataset/model → evaluate → compare → gate → rollback recommendation

Endpoints

  • POST /evaluate
  • POST /compare
  • POST /gate
  • GET /traces
  • POST /packs/compare-gate

What this dashboard is for

  • version comparison
  • latency / correctness monitoring
  • alerting on behavior drift
  • rollback recommendation logic
  • trace view for prompts / responses / failures
  • benchmark pack reuse across releases
