# PyTorch Mini Evaluation Experiment

This is a lightweight PyTorch evaluation demo for FairEval.

It demonstrates:
- tensor-based similarity scoring
- embedding-style comparison
- prompt regression comparison
- constraint failure detection

## What this proves
Hands-on familiarity with PyTorch-based evaluation workflows.

## What this does not prove
- production ML training
- transformer research
- large-scale inference
- GPU optimization
- fine-tuning expertise
