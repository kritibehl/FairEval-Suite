# Embedding Distance Examples

FairEval includes simple prompt and embedding-distance analysis to support NLP evaluation workflows.

## What is measured
- token count estimate
- prompt size distribution
- cosine similarity
- embedding distance

## Why this matters
Prompt size and embedding drift can affect model-output behavior, evaluation stability, and regression analysis.

## Safe scope
This is a lightweight analysis artifact. It does not claim production tokenizer monitoring or large-scale embedding infrastructure.
